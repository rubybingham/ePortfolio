# ePortfolio — Simple scaffold

This file documents the minimal portfolio scaffold added to this workspace.

Files added by the scaffold:

- `index.html` — main page with About, Projects, Video, and Contact sections
- `styles.css` — simple responsive styles
- `script.js` — tiny helper script
- `assets/` — folder for your local video (`assets/video.mp4`) and optional poster (`assets/poster.jpg`)

Instructions:
1. Add your local video at `assets/video.mp4` (or update the `<source>` element in `index.html`).
2. Open `index.html` in a browser or run a local server (recommended).

Quick server (PowerShell):

```powershell
python -m http.server 8000
# then open http://localhost:8000
```
